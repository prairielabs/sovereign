# TOOLS

Operator-specific tool notes. Optional. Sovereign does not require it.

Use this file for environment-specific facts that aren't part of the
constitution: SSH aliases, device names, camera locations, network
shortcuts. Anything that belongs to *your* setup, not to Sovereign.

Skills (in `~/.openclaw/workspace/skills/` if running on OpenClaw)
define *how* tools work. This file is for *which* tools you have.
