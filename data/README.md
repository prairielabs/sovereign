## Reproducing

Every artifact here was produced by the system in this repo. To make
your own:

1. Install per `INSTALLER.md`.
2. Drop a problem in `loop/problems/`.
3. Watch the loop run.
4. The output lands in `loop/results/` and the metric lands in
   `memory/metrics.jsonl`.

The artifact is the proof.
